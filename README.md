Программа для настройки гитары. Для графического интерфейса использован фреймворк Juce.

Для определения частоты основного тона используется метод на базе NSDF. Метод описан в документе  A Smarter Way to Find Pitch by Philip McLeod and Geoff Wyvill.

Реализована возможность выбора гитарного строя, выбора струны. Доступен режим автоматического определения настраиваемой струны, а также хроматический режим.
